# Books

**CashlessConsumer reference books** — definitive, independently researched works on fintech, digital public infrastructure, and consumer finance.

Each book lives in `books/<slug>/` as a set of per-chapter markdown files. A CI pipeline builds PDF, EPUB, MOBI, and HTML on every push to `main` and publishes them to GitHub Pages.

**Browse the library:** https://ccagentorg.github.io/books

---

## Quick Start (for contributors)

```bash
# Clone
git clone https://github.com/CCAgentOrg/books.git
cd books

# Build one book (all formats)
bash scripts/build-book.sh books/aadhaar-esign

# Build all books
bash scripts/build-all.sh
```

Outputs land in `output/`. The library website lives in `docs/`.

## Creating a New Book

1. **Create a branch:** `book/<slug>` (e.g. `book/upi-101`)
2. **Add files:**
   - `books/<slug>/metadata.yaml` — book metadata
   - `books/<slug>/chapter-index.yml` — ordered chapter list
   - `books/<slug>/manuscript/` — per-chapter markdown files
   - `books/<slug>/AGENTS.md` — book-specific agent instructions
   - `books/<slug>/README.md` — contributor guide for this book
3. **Build locally:** `bash scripts/build-book.sh books/<slug>`
4. **Open a PR** to `main`
5. **Merge** → CI builds all formats → deploys to Pages

See `AGENTS.md` at repo root for full conventions on structure, voice, citations, and quality checklist.

## Output Formats

| Format | Extension | Primary Tool |
|--------|-----------|-------------|
| PDF | `.pdf` | pandoc + Eisvogel (XeLaTeX) |
| EPUB | `.epub` | pandoc |
| MOBI | `.mobi` | calibre (from EPUB) |
| HTML | `.html` | pandoc + custom CSS |
| Markdown | `.md` | source (concatenated) |
